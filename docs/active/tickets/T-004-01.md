## Context

The main loop ties everything together: terminal setup, graphics protocol detection, git poller spawn, render loop at target FPS, input handling, and clean shutdown. This is the final integration pass.

## Acceptance Criteria

- [ ] `Picker::from_query_stdio()` detects terminal graphics protocol at startup
- [ ] Pixel dimensions computed from terminal cell size × terminal dimensions
- [ ] Git poller spawned on background thread before entering render loop
- [ ] `RepoSeed` computed once before render loop
- [ ] Main loop: drain git channel → update world → render frame → display via ratatui-image → handle input
- [ ] Frame timing: sleep for remaining budget after render to hit target FPS
- [ ] `dt` clamped at 0.05s to prevent spiral of death
- [ ] 'q' and Esc keys exit cleanly
- [ ] Terminal resize events trigger framebuffer reallocation
- [ ] `ratatui::restore()` called on both normal exit and error paths
- [ ] CLI args parsed via clap with all documented options working
