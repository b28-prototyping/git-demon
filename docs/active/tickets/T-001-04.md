## Context

Left and right verge silhouettes are rendered as filled shapes against the sky, using 2D simplex noise for a natural heightmap. The terrain character is derived from `RepoSeed::terrain_roughness`.

## Acceptance Criteria

- [ ] `draw_terrain` renders left and right verge silhouettes in the sky region above the horizon
- [ ] Terrain height is computed from `noise::OpenSimplex` at `(x * TERRAIN_FREQ, world_time * TERRAIN_DRIFT)`
- [ ] Left and right terrain use different noise seeds (offset) so they are never identical
- [ ] Max terrain height scales with `seed.terrain_roughness`
- [ ] Left terrain uses a cool shadow color, right terrain uses a slightly warmer tint
- [ ] Terrain silhouettes are filled (every pixel from terrain top to horizon)
- [ ] Terrain drifts smoothly over time via the `TERRAIN_DRIFT` parameter
