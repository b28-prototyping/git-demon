## Context

Four post-processing effects create the arcade-cabinet CRT feel: motion blur (blends previous frame), scanline filter (darkens alternate rows), bloom (additive glow on bright pixels), and speed lines (radial streaks at high velocity tiers).

## Acceptance Criteria

- [ ] **Motion blur**: blends current frame with previous at alpha `lerp(0.15, 0.35, speed/12.0)`; disabled by `--no-blur`
- [ ] **Scanline filter**: darkens every even row by 20%; disabled by `--no-scanlines`
- [ ] **Bloom**: identifies pixels above luminance threshold 0.72, applies 3×3 additive blur at `BLOOM_STRENGTH` 0.3; disabled by `--no-bloom`
- [ ] **Speed lines**: at Demon+ tier, `N = (cpm * 8).min(64)` lines radiate from vanishing point `(cx, horizon_y)` in accent color; alpha 80 at Demon, 140 at VelocityDemon
- [ ] Effects apply in correct order: speed lines → motion blur → scanlines → bloom
- [ ] Previous frame buffer is correctly swapped each frame for motion blur
- [ ] All pixel values clamp at 255 (no overflow)
