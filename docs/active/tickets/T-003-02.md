## Context

The HUD is a fixed 18px strip at the bottom of the framebuffer showing live stats: sector number, commits/min, lines added/deleted, files changed, velocity tier badge, and repo name. The tier badge color-codes and strobes at VelocityDemon.

## Acceptance Criteria

- [ ] HUD renders as an 18px strip at frame bottom with 80% alpha black background
- [ ] Content rendered using the bitmap font at 1× scale
- [ ] Fields displayed: SECTOR N, X.X c/min, +N -N, N files, tier badge, repo: name
- [ ] Sector is `total_commits / 100`
- [ ] Tier badge colors: Flatline=dim gray, Cruise=white, Active=cyan, Demon=orange, VelocityDemon=red↔white strobe at 4Hz
- [ ] Repo name is right-aligned
- [ ] HUD hidden when `--no-hud` flag is set
- [ ] Background blends correctly with existing framebuffer content (alpha compositing)
