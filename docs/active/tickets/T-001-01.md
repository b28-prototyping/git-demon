## Context

The builtin 5×7 bitmap font is the only text rendering system in git-demon. It's used by billboard text, HUD overlay, TierGate names, and VelocitySign numbers. The current `font.rs` has the structure but the font data array is all zeros — no glyphs render.

## Acceptance Criteria

- [ ] `FONT_DATA` contains correct 5×7 bitmap data for ASCII 32–126 (95 glyphs)
- [ ] At minimum: digits 0–9, uppercase A–Z, lowercase a–z, space, and punctuation `.:/-+!?`
- [ ] `draw_char` renders a single glyph at specified position, color, and scale (1× or 2×)
- [ ] `draw_text` renders a string with correct character spacing
- [ ] `text_width` returns accurate pixel width for a given string and scale
- [ ] No file I/O — font is a compile-time constant
- [ ] Unit tests verify at least 3 glyphs render correct pixel patterns
