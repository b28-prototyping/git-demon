## Sprites and HUD

Implement the depth-projected sprite renderer for roadside objects (billboards, towers, shards, tier gates) and the bottom HUD overlay strip showing live stats.

### Track Breakdown

Both tickets depend on the font (T-001-01) and road rasterizer (T-001-03) from S-001.

- T-003-01: Sprite rendering system — projects world objects to screen
- T-003-02: HUD overlay — bottom strip with sector, c/min, tier badge

These two can run in parallel since they write to different screen regions.

### DAG

```
T-001-01 (font) ────┬──▶ T-003-01 (sprites)
T-001-03 (road) ────┤
                     └──▶ T-003-02 (hud)
```
