## Road Grid Scrolling Fix

Grid lines on the road surface are rendered at fixed world positions that don't move with the camera, causing a Zeno's paradox effect where lines approach but never pass through the viewport. This makes lines accumulate visually and breaks the forward-motion illusion.

### Root Cause

`draw_grid` in `src/renderer/road.rs` places horizontal grid lines at `z_world = i * grid_spacing` (fixed world coordinates). The road stripes correctly scroll using `z_offset / depth`, but the grid ignores `camera_z` entirely.

### Track Breakdown

- T-005-01: Fix grid line scrolling to be camera-relative
