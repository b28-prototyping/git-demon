## Core Rendering Pipeline

Get the fundamental rendering loop working end-to-end: bitmap font, sky, road with curvature, terrain silhouettes, and the effect passes (bloom, motion blur, scanlines). After this story completes, `cargo run` should show a visually complete animated highway scene even without git data.

### Track Breakdown

**Track A (independent):**
- T-001-01: Bitmap font — no dependencies, needed by HUD and sprites
- T-001-02: Sky + sun rendering — no dependencies

**Track B (sequential):**
- T-001-03: Road rasterizer with curvature — no dependencies
- T-001-04: Terrain silhouettes — no dependencies

**Track C (depends on A+B):**
- T-001-05: Effects passes (bloom, motion blur, scanlines, speed lines) — depends on road + sky being renderable

### DAG

```
T-001-01 (font)  ──────────────────┐
T-001-02 (sky)   ──────────────────┤
T-001-03 (road)  ──────────────────┼──▶ T-001-05 (effects)
T-001-04 (terrain) ────────────────┘
```
